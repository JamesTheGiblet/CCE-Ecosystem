## FAQ

### Q: Is this a paper trading simulator?
**A:** Yes and no.  

It's not a "practice trading" tool.  
It's a **proof system** that shows you how the CCE behaves under real market conditions.  

The engine uses the same logic as Node, but doesn't execute trades.

---

### Q: Can I use Cloud to test my own strategies?
**A:** No.  

Cloud runs the CCE strategy only.  
It's designed to build trust in the CCE philosophy, not as a generic backtesting platform.

---

### Q: Does Cloud guarantee Node will perform the same way?
**A:** No.  

Cloud shows how the engine **behaves**.

Cloud is transparency, not a performance promise.
